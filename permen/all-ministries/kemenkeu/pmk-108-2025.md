## BAB I - KETENTUAN UMUM

### Pasal 1
Dalam Peraturan Menteri ini yang dimaksud dengan:

1. **Teknologi Blockchain** adalah teknologi distributed ledger yang menggunakan cryptographic hash untuk menciptakan rangkaian blok data yang immutable dan transparan.

2. **Distributed Ledger Technology (DLT)** adalah sistem database terdistribusi yang memungkinkan penyimpanan dan verifikasi data secara desentralisasi.

3. **Smart Contract** adalah protokol komputer yang secara otomatis mengeksekusi, mengontrol, atau mendokumentasikan peristiwa dan tindakan sesuai ketentuan kontrak.

4. **Cryptographic Hash** adalah fungsi matematika yang mengubah input data menjadi string karakter dengan panjang tetap yang bersifat deterministik dan tidak dapat dibalik.

5. **Consensus Mechanism** adalah protokol yang digunakan untuk mencapai kesepakatan tentang kebenaran data dalam jaringan blockchain.

6. **Node** adalah komputer atau server yang berpartisipasi dalam jaringan blockchain dengan menyimpan salinan ledger.

7. **Private Blockchain** adalah jaringan blockchain yang aksesnya terbatas pada organisasi atau grup tertentu.

8. **Hybrid Blockchain** adalah kombinasi antara public dan private blockchain dengan kontrol akses yang dapat dikonfigurasi.

contract BudgetExecution {
    mapping(address => uint256) public budgetAllocation;
    mapping(address => uint256) public expenditure;
    
    function allocateBudget(address ministry, uint256 amount) 
        onlyAuthorized {
        budgetAllocation[ministry] = amount;
        emit BudgetAllocated(ministry, amount);
    }
    
    function executePayment(uint256 amount, string memory purpose) 
        onlyMinistry {
        require(expenditure[msg.sender] + amount <= 
                budgetAllocation[msg.sender]);
        expenditure[msg.sender] += amount;
        emit PaymentExecuted(msg.sender, amount, purpose);
    }
}

        onlyMinistry public {
        require(expenditure[msg.sender] + amount <= 
                budgetAllocation[msg.sender], "Budget exceeded");
        expenditure[msg.sender] += amount;
        emit PaymentExecuted(msg.sender, amount, purpose, block.timestamp);
    }
}
