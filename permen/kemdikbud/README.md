# Permendikbud Regulations Database

This directory contains individual Permendikbud (Ministry of Education, Culture, Research, and Technology) regulations converted to markdown format.

## File Naming Convention

`permendikbud-[number]-[year]-[topic].md`

## Priority Categories Covered

1. **Merdeka Belajar Program (5-8 regulations)**
   - Curriculum flexibility
   - Assessment system modernization
   - Student-centered learning approaches

2. **Higher Education Regulations (8-10 regulations)**
   - University autonomy and accreditation
   - Research and innovation frameworks
   - International collaboration standards

3. **Basic Education Standards (6-8 regulations)**
   - National curriculum standards
   - Teacher certification requirements
   - School quality assurance systems

4. **Cultural Preservation (4-6 regulations)**
   - Traditional arts and heritage protection
   - Museum and cultural site management
   - Cultural education integration

5. **Technology in Education (4-6 regulations)**
   - Digital learning platform standards
   - Educational technology integration
   - Online learning quality assurance

## Content Structure

Each regulation file contains:
- ✅ Complete legal text in clean markdown
- ✅ Educational impact analysis
- ✅ Practical application guidelines
